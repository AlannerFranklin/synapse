# Synapse

A lightweight, high-performance Go LLM Agent framework focused on parallel execution (DAG) and advanced memory management.

## Features (Planned)
- 🚀 **Parallel Execution**: Native support for DAG-based parallel task execution using Go's concurrency.
- 🧠 **Advanced Memory**: Built-in support for short-term sliding windows and long-term semantic memory (RAG).
- 🔌 **Lightweight**: Minimal abstractions, easy to customize and extend.
